# template-name

> my transcendent SAO template

## Usage

Install [SAO](https://github.com/egoist/sao) first.

```bash
yarn global add sao
# or
npm i -g sao
```

### From npm

```bash
sao name my-project
```

### From git

```bash
sao yuri parsons/template-name my-project
```
